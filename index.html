Electron DaisyUI Framework

基于 Electron + DaisyUI + Tailwind CSS 的现代桌面应用开发框架。 快速构建美观、响应式的桌面应用程序。

跨平台桌面应用

基于 Electron 构建，支持 Windows、macOS 和 Linux 三大平台

现代化界面设计

集成 DaisyUI 和 Tailwind CSS，提供丰富的主题和组件

快速开发

提供完整的模板系统和开发工具，让你专注于业务逻辑

快速入门

1. 开发环境设置

npm install
npm run dev

2. 创建新页面

复制页面模板
添加到路由配置
自定义页面内容

3. 项目结构

src/renderer/pages/ - 页面文件
src/renderer/components/ - 组件文件
templates/ - 页面和组件模板
main.js - Electron 主进程
tailwind.config.js - 样式配置
